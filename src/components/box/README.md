### Primary
```jsx
<div style={{ display: 'flex' }}>
  <Box content="Button" appearance="primary" style={{ marginRight: 10 }} />
  <Box content="Disabled" appearance="primary" disabled />
</div>
```

### Default
```jsx
<div style={{ display: 'flex' }}>

  <Box content="Disabled" appearance="default" disabled />
</div>
```
